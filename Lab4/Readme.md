# Lab Report Sensor Devices Used in IoT Projects

## Objective

* To study commonly used sensor devices in IoT systems
* To understand working principles of different sensors and modules
* To identify applications of sensors in real world IoT projects
* To learn hardware components used in project development

---

## Introduction

Internet of Things systems connect physical devices to the internet so they can collect and exchange data. Sensors are the main input devices in IoT systems. They detect changes in the environment and send signals to microcontrollers like ESP32 or Arduino for processing.

This lab focuses on commonly used sensor devices and electronic components used in student IoT projects.

---

## Components Studied

### ESP32 Development Board

**Example Device:** ESP32 DevKit V1

ESP32 is a microcontroller with built in WiFi and Bluetooth used as the main controller in IoT projects.

**Applications**

* Smart home automation
* IoT monitoring systems
* Wireless communication systems
* Safety and tracking devices

---

### Breadboard

**Example Device:** MB-102 Breadboard

A breadboard is used for building circuits without soldering.

**Applications**

* Circuit prototyping
* Testing electronic components
* Educational experiments

---

### Jumper Wires

**Example Device:** Male to Male Jumper Wires

Used to connect components in a circuit.

**Applications**

* Sensor connections
* Circuit assembly
* Prototype development

---

### Obstacle Detection Sensor IR Sensor

**Example Device:** FC-51 IR Sensor

Detects obstacles using infrared light reflection.

**Applications**

* Obstacle avoidance robot
* Line following robot
* Smart vehicles

---

### Ultrasonic Distance Sensor

**Example Device:** HC-SR04 Ultrasonic Sensor

Measures distance using sound waves.

**Applications**

* Smart parking system
* Distance measurement

---

### Camera Module for Face Detection

**Example Device:** ESP32 CAM Module OV2640

Used for image capture and face detection.

**Applications**

* Attendance system
* Security surveillance
* Face recognition projects

### Motion Detection Sensor

**Example Device:** HC-SR501 PIR Sensor

* Security alarms

## Procedure

1. Collect all sensor modules and components
2. Identify each device and study its function
3. Connect sensors using breadboard and jumper wires
4. Interface sensors with ESP32 microcontroller
5. Upload sample programs for testing
6. Observe output in serial monitor
7. Record results and applications
